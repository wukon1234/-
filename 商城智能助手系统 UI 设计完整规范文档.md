# 商城智能助手系统 UI 设计完整规范文档

**文档说明**：本规范为商城智能助手系统全平台UI设计标准，包含视觉体系、组件规范、页面结构、交互动效、响应式规则、前端开发适配标准，可直接用于设计还原、前端开发、项目验收，整体风格为轻奢科技风，突出AI智能助手核心能力与电商购物体验。

# 一、项目概述

本项目为融合电商购物与AI智能对话的一体化商城智能助手系统，核心覆盖商品浏览、选购、加购、订单管理、智能问答、商品推荐、购物引导全场景业务。整体UI采用轻奢科技设计语言，以统一的视觉规范、细腻的交互体验、严谨的布局逻辑，打造高端、精致、易用、符合现代用户审美的商业级产品界面，同时完美适配PC端、平板端、移动端全终端设备。

# 二、全局视觉设计规范

## 2\.1 色彩系统

|色彩类型|色值/渐变|应用场景|
|---|---|---|
|主色调|\#667eea → \#764ba2 蓝紫渐变|品牌标识、主按钮、导航激活态、聊天气泡、核心强调、边框高亮|
|中性背景色|\#ffffff、\#f8f9fa、\#f1f3f5|页面背景、卡片底色、面板区域、分割区域|
|中性文字色|\#222222、\#444444、\#888888|标题、正文、辅助说明、提示文案|
|功能状态色|成功绿\#52c41a、警告橙\#faad14、错误红\#ff4d4f、提示蓝\#1890ff|表单验证、操作反馈、订单状态、系统提示|

**色彩原则**：低饱和度、高通透度、渐变柔和不艳俗，全系统配色统一，文本对比度符合可访问性标准，无杂乱视觉干扰。

## 2\.2 字体排版规范

全局采用现代无衬线字体：Inter、PingFang SC、Microsoft YaHei、Roboto，字重层级严格统一，保证全平台阅读舒适度。

- 主标题：24\-36px，字重700，用于页面大标题、Banner标题、核心数据展示

- 副标题：18\-22px，字重600，用于模块标题、弹窗标题、分类标题

- 正文内容：14\-16px，字重400\-500，用于商品描述、对话内容、表格文本

- 辅助文字：12px，字重400，用于标签、时间、提示语、备注、版权信息

## 2\.3 布局与基础视觉规范

- 栅格系统：全局采用8px基础栅格，所有间距、尺寸、内边距严格遵循栅格规则

- 留白规范：大面积呼吸感留白，模块间距统一为20px、24px、32px三级标准

- 圆角规范：基础卡片12px，按钮8px，弹窗16px，消息气泡异形圆角，全局统一柔和质感

- 阴影规范：多层轻柔投影，近实远虚，拒绝厚重脏阴影，hover状态增强阴影提升层级

- 过渡动画：全局统一0\.25s\-0\.3s ease\-out流畅微动效，轻盈不拖沓，无生硬跳转

# 八、设计原则与最终目标

## 8\.1 核心设计原则

- **一致性**：全系统视觉语言、组件样式、交互逻辑完全统一

- **极简克制**：去除所有冗余装饰，突出核心业务内容，减少视觉噪音

- **层级清晰**：通过阴影、色彩、间距构建明确的视觉层级，用户可快速抓取重点

- **高可用性**：操作路径直观简洁，降低用户学习成本，提升购物效率

- **可访问性**：保证文本高对比度，合理触控区域，适配全用户群体

## 8\.2 设计最终目标

- 打造高端、精致、现代、富有质感的商城智能助手UI界面

- 实现流畅舒适、反馈及时的全链路交互体验

- 突出AI智能助手核心功能，打造电商\+AI的完整使用闭环

- 全终端完美适配，保证多设备体验一致性

- 达到商业级上线产品视觉标准，提升用户留存、转化率与整体满意度
