# 商城项目功能介绍文档

## 1. 项目概述

本项目是一个基于 Spring Boot 和 Vue3 开发的现代化商城系统，集成了智能助手功能，旨在提供一站式的在线购物体验。系统采用前后端分离架构，后端使用 Spring Boot 构建 RESTful API，前端使用 Vue3 + TypeScript 开发响应式界面，同时集成了大模型技术，为用户提供智能购物助手服务。

### 1.1 项目目标

- 提供稳定的商品浏览、搜索、下单与支付流程
- 集成智能助手，提升用户购物体验和客服效率
- 支持多角色用户管理，包括普通用户、管理员和客服人员
- 实现高性能、高可用的系统架构
- 引入 RAG（检索增强生成）技术，提升智能助手的回答准确性
- 支持流式对话，提供实时的用户交互体验

## 3. 核心功能模块

### 3.1 用户管理

#### 3.1.1 注册与登录
- 支持用户注册，包括手机号、邮箱等方式
- 实现密码加密存储
- 支持 JWT 认证登录
- 提供密码找回功能

#### 3.1.2 个人中心
- 查看个人信息
- 修改个人资料
- 管理收货地址
- 查看订单历史
- 管理收藏列表

### 3.2 商品管理

#### 3.2.1 商品浏览
- **商品列表展示**：支持分页，每页显示指定数量的商品
- **商品分类筛选**：根据商品分类进行筛选
- **商品排序**：支持按创建时间、价格等排序
- **商品搜索**：支持关键字搜索，搜索范围包括商品名称、描述、规格等

#### 3.2.2 商品详情
- **商品图片**：支持商品图片展示，图片加载失败时显示默认图标
- **商品信息**：展示商品名称、价格、描述、规格等信息
- **商品状态**：只显示上架状态的商品
- **相关推荐**：基于商品信息推荐相关商品

#### 3.2.3 后台商品管理
- **商品添加**：添加新商品，包括名称、价格、描述、规格、分类等信息
- **商品编辑**：修改现有商品信息
- **商品删除**：删除不需要的商品
- **商品上下架**：修改商品状态，控制商品是否显示
- **商品库存管理**：管理商品库存数量
- **商品分类管理**：管理商品分类

#### 3.2.4 商品文案生成
- **AI 辅助生成**：基于大模型生成商品标题、简介、详细描述、营销卖点
- **定制化生成**：根据商品名称、关键词、目标人群生成符合商城调性的文案
- **快速编辑**：生成后可进行人工编辑调整

### 3.3 购物车管理

- 添加商品到购物车
- 修改购物车商品数量
- 删除购物车商品
- 购物车结算
- 促销信息展示

### 3.4 订单管理

#### 3.4.1 下单流程

- 选择支付方式
- 确认订单信息
- 提交订单

#### 3.4.2 订单状态管理

- 已付款

- 已取消

#### 3.4.3 后台订单管理

- 订单详情查看
- 订单状态修改
- 订单发货管理
- 订单统计分析

### 3.5 地址管理

- 添加收货地址
- 编辑收货地址

### 3.6 评价系统

- 商品评价（文字、图片）

### 3.7 收藏系统

- 商品收藏
- 收藏列表管理
- 取消收藏

### 3.8 智能助手

#### 3.8.1 核心功能
- **商品问答**：回答商品相关问题，如价格、规格、库存等
- **订单查询**：查询订单状态和物流信息
- **商品推荐**：基于用户需求推荐商品，支持直接跳转商品详情页
- **多轮对话**：保持会话上下文，支持连续对话
- **流式对话**：实时显示回复内容，提升用户体验
- **商品文案生成**：基于商品信息生成吸引人的商品描述、标题等
- **RAG调试**：提供调试接口，查看检索效果

#### 3.8.2 技术实现
- **大模型集成**：基于通义千问大模型，支持多模型提供商配置
- **RAG技术**：集成检索增强生成技术，提高回答准确性
- **向量数据库**：使用 Chroma 作为向量存储，支持商品信息的语义检索
- **流式对话**：基于 SSE（Server-Sent Events）实现实时对话
- **会话管理**：完整的会话历史记录和管理功能
- **商品推荐卡片**：前端实现商品推荐卡片，支持直接跳转

#### 3.8.3 会话管理
- **会话历史记录**：保存用户与智能助手的对话历史
- **会话列表管理**：展示用户的所有会话，支持分页
- **会话删除功能**：允许用户删除不需要的会话
- **会话状态管理**：保持会话上下文，支持多轮对话

#### 3.8.4 API 接口
- **POST /api/assistant/chat**：发送消息并获取回复
- **POST /api/assistant/chat/stream**：流式发送消息，实时获取回复
- **GET /api/assistant/conversations**：获取会话列表
- **GET /api/assistant/conversation/{sessionId}/messages**：获取会话历史消息
- **DELETE /api/assistant/conversation/{sessionId}**：删除会话
- **POST /api/assistant/rag/sync/products**：同步商品向量到向量库
- **GET /api/assistant/rag/debug/search**：RAG调试接口，查看检索效果

## 4. 智能助手详细功能

### 4.1 功能特性

| 功能 | 描述 | 实现方式 |
|------|------|----------|
| 商品问答 | 回答商品相关问题，如价格、规格、库存等 | 基于 RAG 检索商品信息，结合大模型生成回答 |
| 订单查询 | 查询用户订单状态、物流信息 | 基于用户授权，检索订单数据库 |
| 商品推荐 | 根据用户需求推荐相关商品 | 结合用户历史行为和会话上下文 |
| 多轮对话 | 保持会话上下文，支持连续对话 | 会话状态管理和上下文压缩 |
| 流式输出 | 实时显示回复内容，提升用户体验 | Server-Sent Events 或 WebSocket |
| 商品卡片 | 推荐商品以卡片形式展示，支持直接跳转 | 前端组件渲染 |

### 4.2 技术架构

#### 4.2.1 后端实现
- **AssistantService**：智能助手核心服务，处理聊天请求和会话管理
- **LLMService**：大模型服务，负责与通义千问 API 交互
- **RAGService**：检索增强生成服务，整合向量检索和大模型生成
- **EmbeddingService**：文本向量化服务，将商品信息转换为向量
- **DataSource**：数据源接口，支持商品、订单等数据的检索
- **AssistantController**：智能助手控制器，处理 HTTP 请求和 SSE 流式响应

#### 4.2.2 前端实现
- **Assistant.vue**：智能助手主页面，包含会话列表和聊天窗口
- **ChatWindow.vue**：聊天窗口组件，支持流式对话、消息展示和商品推荐卡片
- **ConversationList.vue**：会话列表组件，展示用户的所有会话
- **MessageInput.vue**：消息输入组件，支持用户输入消息
- **ProductRecommendation.vue**：商品推荐卡片组件，展示推荐商品，支持点击跳转

#### 4.2.3 前端组件详细实现
- **Assistant.vue**：
  - 采用响应式布局，左侧会话列表，右侧聊天窗口
  - 支持会话切换、新建会话、删除会话
  - 实时加载会话列表和消息历史

- **ChatWindow.vue**：
  - 支持普通对话和流式对话模式切换
  - 实时显示流式回复内容，带有加载动画
  - 展示用户和助手的消息，区分不同角色
  - 集成商品推荐卡片组件
  - 自动滚动到底部，确保最新消息可见

- **ProductRecommendation.vue**：
  - 展示推荐商品列表，每个商品以卡片形式呈现
  - 包含商品图片、名称、价格等信息
  - 支持点击商品卡片跳转到商品详情页
  - 图片加载失败时显示默认图标
  - 响应式布局，适配不同屏幕尺寸

### 4.3 工作流程

1. 用户发送消息到智能助手
2. 后端接收消息，进行意图识别
3. 根据意图选择相应的处理策略
   - 商品问答：检索商品信息，生成回答
   - 订单查询：查询订单数据库，返回结果
   - 商品推荐：分析用户需求，推荐相关商品
4. 生成回复，通过流式或普通方式返回给前端
5. 前端展示回复内容，包括文字和商品推荐卡片
6. 保存会话历史，支持后续对话
