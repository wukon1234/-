# 功能实现说明

## 已完成功能

### 1. JWT用户认证 ✅

#### 后端实现
- **JwtUtil**: JWT工具类，提供Token生成、验证、解析功能
- **JwtInterceptor**: JWT拦截器，自动验证请求中的Token
- **WebConfig**: Web配置类，注册拦截器
- **UserService**: 用户服务，实现登录、注册功能
- **UserController**: 用户控制器，提供登录、注册API

#### 功能特性
- 用户注册（密码BCrypt加密）
- 用户登录（生成JWT Token）
- Token自动验证（拦截器）
- 支持从Token中提取用户信息

#### API接口
```
POST /api/user/register - 用户注册
POST /api/user/login - 用户登录
GET /api/user/info - 获取当前用户信息
```

### 2. 流式对话功能（SSE）✅

#### 后端实现
- **AssistantController.chatStream**: 流式对话接口，使用SSE（Server-Sent Events）
- **AssistantServiceImpl.chatStream**: 流式对话服务实现
- **QwenLLMServiceImpl.chatStream**: 通义千问流式调用（模拟流式输出）

#### 功能特性
- 支持实时流式输出（逐字显示）
- 使用SSE协议实现服务器推送
- 异步处理，不阻塞请求
- 支持错误处理和超时控制

#### API接口
```
POST /api/assistant/chat/stream - 流式对话（SSE）
Content-Type: text/event-stream
```

#### 前端实现
- **chatStreamPost**: 使用fetch + ReadableStream实现流式接收
- **ChatWindow组件**: 支持流式对话开关
- 实时显示流式输出内容

### 3. 商品推荐功能 ✅

#### 后端实现
- **RAGService**: RAG检索服务，根据用户问题检索相关商品
- **AssistantService**: 在对话过程中自动检索相关商品
- **ChatResponse**: 响应中包含推荐商品列表
- **StreamCallback.onProducts**: 流式对话完成后发送商品推荐

#### 功能特性
- 根据用户问题语义检索相关商品
- 返回Top-K个最相关的商品（当前K=5）
- 商品信息包含：名称、价格、图片、描述

#### 前端实现
- **ProductRecommendation组件**: 商品推荐展示组件
- **ChatWindow组件**: 在消息中展示推荐商品
- 商品卡片支持点击跳转到商品详情页

#### 展示方式
- 在助手回复消息下方显示推荐商品卡片
- 商品卡片包含图片、名称、价格
- 支持点击查看商品详情

## 使用说明

### 1. 用户认证

#### 注册
```javascript
POST /api/user/register
{
  "username": "testuser",
  "password": "123456",
  "email": "test@example.com"
}
```

#### 登录
```javascript
POST /api/user/login
{
  "username": "testuser",
  "password": "123456"
}

Response:
{
  "code": 200,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "userId": 1,
    "username": "testuser"
  }
}
```

#### 使用Token
```javascript
// 在请求头中携带Token
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### 2. 流式对话

#### 开启流式对话
在前端ChatWindow组件中，默认开启流式对话模式。

#### 请求示例
```javascript
POST /api/assistant/chat/stream
Content-Type: application/json

{
  "sessionId": "会话ID（可选）",
  "message": "我想要一部手机"
}

// 响应为SSE流
data: 我为您推荐
data: 以下手机
data: ...
event: products
data: {"products": [...]}
event: done
data: [DONE]
```

### 3. 商品推荐

商品推荐会自动在对话过程中触发：
1. 用户发送消息后，系统使用RAG检索相关商品
2. 检索到的商品信息会被包含在助手回复中
3. 前端自动在消息下方展示商品推荐卡片

## 技术要点

### JWT认证流程
1. 用户登录 → 生成JWT Token
2. 客户端保存Token（localStorage）
3. 每次请求携带Token在请求头
4. 后端拦截器验证Token
5. 验证通过后，将用户信息放入request attribute

### 流式对话实现
1. 前端使用fetch + ReadableStream接收SSE流
2. 后端使用SseEmitter发送流式数据
3. 大模型响应分块发送
4. 前端实时渲染流式内容

### RAG商品检索
1. 用户问题向量化（当前使用关键词匹配，后续可升级为向量检索）
2. 在商品数据库中检索相似商品
3. 返回Top-K个最相关商品
4. 将商品信息作为上下文提供给大模型

### 4. 购物车功能 ✅

#### 后端实现
- **Cart实体类**: 购物车实体
- **CartMapper**: 购物车数据访问层
- **CartService**: 购物车服务，实现添加、查询、更新、删除等功能

```
POST /api/cart/add - 添加商品到购物车
GET /api/cart/list - 获取购物车列表
PUT /api/cart/{cartId} - 更新购物车商品数量
DELETE /api/cart/{cartId} - 删除购物车商品
DELETE /api/cart/clear - 清空购物车
```

#### 前端实现
- **Cart.vue**: 购物车页面，支持全选、批量删除、数量修改、结算等功能
- **cart.ts**: 购物车API接口封装

### 5. 订单功能 ✅

#### 后端实现
- **Order实体类**: 订单实体
- **OrderItem实体类**: 订单项实体
- **OrderMapper、OrderItemMapper**: 订单数据访问层
- **OrderService**: 订单服务，实现创建、查询、取消、支付等功能
- **OrderController**: 订单控制器，提供订单相关API

#### 功能特性
- 创建订单（从购物车创建，自动扣减库存）
- 查询订单列表（支持按状态筛选）
- 查询订单详情
- 取消订单（恢复库存）
- 支付订单（模拟支付）

#### API接口
```
POST /api/order/create - 创建订单
GET /api/order/list - 获取订单列表（支持status参数筛选）
GET /api/order/{orderId} - 获取订单详情
PUT /api/order/{orderId}/cancel - 取消订单
PUT /api/order/{orderId}/pay - 支付订单
```

#### 订单状态
- 0: 待支付
- 1: 已支付
- 2: 已发货
- 3: 已完成
- 4: 已取消

#### 前端实现
- **OrderList.vue**: 订单列表页面，支持状态筛选、订单详情查看、支付、取消等功能
- **order.ts**: 订单API接口封装

### 6. 智能助手会话管理功能 ✅

#### 后端实现
- **AssistantService**: 新增会话列表、消息历史、删除会话等方法
- **AssistantController**: 完善会话管理相关API接口

#### 功能特性
- 获取会话列表（支持分页）
- 获取会话历史消息
- 删除会话（同时删除会话下的所有消息）

```
GET /api/assistant/conversation/{sessionId}/messages - 获取会话历史消息
DELETE /api/assistant/conversation/{sessionId} - 删除会话
```

#### 前端实现
- **Assistant.vue**: 智能助手主页面，整合会话列表和对话窗口
- **ConversationList.vue**: 会话列表组件，支持新建、选择、删除会话
- **ChatWindow.vue**: 对话窗口组件，支持普通对话和流式对话
- **MessageInput.vue**: 消息输入组件
- **ProductRecommendation.vue**: 商品推荐展示组件
