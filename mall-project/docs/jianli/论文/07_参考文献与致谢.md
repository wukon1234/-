# 附录

## 附录A 系统运行环境配置说明

### A.1 环境要求

| 软件 | 版本要求 | 用途 |
|------|----------|------|
| JDK | 17或更高 | Java运行环境 |
| Maven | 3.6或更高 | 项目构建工具 |
| Node.js | 18或更高 | 前端运行环境 |
| MySQL | 8.0 | 业务数据库 |
| Redis | 7.0（可选） | 缓存数据库 |

### A.2 数据库初始化

1. 登录MySQL数据库：
```bash
mysql -u root -p
```

2. 执行数据库初始化脚本：
```bash
source /path/to/docs/数据库脚本.sql
```

3. 脚本将创建mall_db数据库及所有业务表，并插入测试数据。

### A.3 后端服务启动

1. 进入后端项目目录：
```bash
cd mall-backend
```

2. 修改配置文件application.yml，配置数据库连接信息、LLM API密钥等参数。

3. 编译并运行项目：
```bash
mvn clean install
mvn spring-boot:run
```

4. 服务启动后，访问http://localhost:8080可验证后端运行状态。

### A.4 前端应用启动

1. 进入前端项目目录：
```bash
cd mall-frontend
```

2. 安装项目依赖：
```bash
npm install
```

3. 启动开发服务器：
```bash
npm run dev
```

4. 访问http://localhost:5173即可打开系统前端界面。

## 附录B 核心接口文档

### B.1 用户认证接口

**用户登录**
- 请求：POST /api/user/login
- 请求体：{"username": "string", "password": "string"}
- 响应：{"code": 200, "data": {"token": "string", "userId": number, "username": "string"}}

**用户注册**
- 请求：POST /api/user/register
- 请求体：{"username": "string", "password": "string", "email": "string"}
- 响应：{"code": 200, "message": "注册成功"}

### B.2 商品接口

**商品列表**
- 请求：GET /api/products?page=1&size=10&categoryId=1&keyword=手机
- 响应：{"code": 200, "data": {"records": [...], "total": 100, "pages": 10}}

**商品详情**
- 请求：GET /api/products/{id}
- 响应：{"code": 200, "data": {"id": 1, "name": "...", "price": 8999, ...}}

### B.3 智能助手接口

**流式对话**
- 请求：POST /api/assistant/chat/stream
- 请求头：Accept: text/event-stream
- 请求体：{"sessionId": "string", "message": "string"}
- 响应：SSE事件流
