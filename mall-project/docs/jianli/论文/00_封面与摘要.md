## 摘要

伴随电子商务产业的蓬勃发展以及大语言模型技术的持续突破，将智能对话助手融入电商购物场景已成为提升用户消费体验的关键方向。本文针对传统电商平台在商品检索效率、客服响应能力和个性化服务等方面存在的不足，设计并实现了一套基于Spring Boot和Vue3技术框架的前后端分离商城系统，同时集成了基于RAG（检索增强生成）技术的智能购物助手模块。

在系统架构层面，后端服务采用Spring Boot 2.7框架构建，借助MyBatis Plus实现数据持久化操作，使用MySQL 8.0存储业务数据，可选配Redis作为缓存支撑。前端应用采用Vue 3框架开发，结合TypeScript增强代码健壮性，使用Element Plus组件库构建用户界面，通过Pinia进行状态管理。系统采用JWT令牌实现无状态身份认证，使用BCrypt算法加密存储用户密码，保障了系统的安全性。

在智能助手模块设计中，系统对接了大语言模型服务API，通过RAG技术将商品数据库中的商品信息作为检索上下文注入模型提示词，有效提升了回答内容的准确性和业务相关性。系统设计了可扩展的LLM服务接口和RAG数据源接口，支持对接不同的模型供应商和扩展多种业务数据的检索能力。同时，系统采用SSE（服务器推送事件）技术实现对话内容的流式输出，用户可实时观察助手回复的生成过程，显著优化了对话交互的感知体验。

系统实现了完整的电商业务功能闭环，涵盖用户账户注册登录、商品分类浏览、购物车增删改查、订单创建支付取消、收货地址管理等核心模块；同时实现了智能助手的会话管理、多轮对话、商品检索推荐等智能化服务功能。经过系统性的功能测试、性能测试和安全性测试验证，系统各项功能运行稳定，性能指标达到预期目标，具备实际应用价值。

**关键词**：电子商务系统；Spring Boot；Vue3；大语言模型；检索增强生成；智能购物助手

## Abstract

With the vigorous development of the e-commerce industry and the continuous breakthrough of large language model technology, integrating intelligent conversational assistants into e-commerce shopping scenarios has become a key direction for improving user consumption experience. Addressing the deficiencies of traditional e-commerce platforms in product search efficiency, customer service response capability, and personalized services, this paper designs and implements a front-end and back-end separated mall system based on Spring Boot and Vue3 technology frameworks, while integrating an intelligent shopping assistant module based on RAG (Retrieval-Augmented Generation) technology.

At the system architecture level, the backend service is built using the Spring Boot 2.7 framework, utilizing MyBatis Plus for data persistence operations, MySQL 8.0 for storing business data, and optionally configuring Redis as cache support. The frontend application is developed using the Vue 3 framework, combined with TypeScript to enhance code robustness, using the Element Plus component library to build the user interface, and Pinia for state management. The system adopts JWT tokens for stateless identity authentication and uses the BCrypt algorithm to encrypt and store user passwords, ensuring system security.

In the design of the intelligent assistant module, the system interfaces with large language model service APIs, using RAG technology to inject product information from the product database as retrieval context into model prompts, effectively improving the accuracy and business relevance of response content. The system designs extensible LLM service interfaces and RAG data source interfaces, supporting connections to different model providers and extending retrieval capabilities for various business data. Additionally, the system uses SSE (Server-Sent Events) technology to achieve streaming output of conversation content, allowing users to observe the generation process of assistant replies in real-time, significantly optimizing the perceived experience of conversation interaction.

The system implements a complete e-commerce business function loop, covering core modules such as user account registration and login, product category browsing, shopping cart CRUD operations, order creation, payment, and cancellation, and shipping address management. It also implements intelligent service functions including session management, multi-turn dialogue, and product search recommendations for the intelligent assistant. After systematic functional testing, performance testing, and security testing verification, all system functions operate stably, performance indicators meet expected goals, and the system has practical application value.

**Keywords**: E-commerce System; Spring Boot; Vue3; Large Language Model; Retrieval-Augmented Generation; Intelligent Shopping Assistant
